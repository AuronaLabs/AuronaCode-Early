# 产品方向 (Product Direction)

Aurona Code 是一个探索将 AI 辅助编程深度融入本地开发环境的桌面代码编辑器项目

## 核心定位

- **Agent 原生探索**：尝试将大语言模型与代码库上下文（如 Git 历史、文件依赖树）进行结构化绑定
- **现代跨端架构**：基于前端技术栈与 Tauri 构建桌面应用，控制资源占用，提供基础代码编辑能力
- **版本数据驱动**：集成基础的版本控制能力，作为辅助理解代码变更状态的数据源

## 阶段目标

- 建立稳定的本地工作区文件扫描与持久化缓存机制
- 完善 Monaco Editor 的基础语言服务支持
- 探索在本地 IDE 闭环内融合自动化脚本与 AI 会话交互的可行形态
