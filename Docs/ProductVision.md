# 产品愿景 (Product Vision)

Aurona Code 是一个**原生面向智能 Agent (Agent-native)** 的新一代开发环境。它的诞生是为了那些希望编辑器能“真正理解整个项目”而非仅仅“显示代码文件”的开发者。

## 核心定位 (Positioning)

- **Agent-native**：未来的交互不仅局限于侧边栏的聊天框，而是 AI 与代码上下文的深度融合与协同。
- **现代化与年轻化**：抛弃繁杂的传统 IDE 历史包袱，打造一款具有现代审美、流畅体验且足以支撑日常高强度工程开发的趁手工具。
- **以 Git 为核心的第一印象**：理解代码库的基础是理解代码变更。Aurona 的核心设计之一，是通过深入解析版本控制的脉络，帮助开发者梳理修改故事，建立信任。
- **桌面级原生体验**：通过 Tauri 构建，将高保真的前端与底层的系统性能完美结合。

## 核心承诺 (Core Promise)

Aurona Code 将在未来帮助开发者清晰地了解：
- 代码库究竟发生了什么改变？
- 这些改变背后的原因和潜在影响是什么？
- 哪些文件在当前的重构中蕴含最大的风险？
- 一次完美的 Commit 应该讲述怎样的一个故事？
- 当真正的 AI 工具接入后，下一步能够最高效执行的动作是什么？
