# Aurona Code 0.3.1 Release Notes

0.3.1 是基于 0.3.0 用户反馈完成的体验精修版本，不扩张大型功能边界。

## 主要改进

- 编辑器文本、光标、选择、点击命中和虚拟视口改用统一布局度量，改善不同字体、DPI、Tab、中文与 emoji 下的定位准确性，并移除旧标签栏遗留的顶部占位。
- Windows 主窗口启用 HTML5 文件树拖放，支持单个文件或目录在工作区内移动与复制，并拦截非法目标。
- 首次打开终端面板时自动创建默认 Shell，并避免并发打开产生重复终端。
- Tooltip、Switch 与 Select 完成 Material 材质收口，Tooltip 改为无箭头圆角矩形；标签栏直接继承编辑器卡片的稳定 Panel 材质。
- 全局搜索工具 Fliuno 首先接入命令搜索，采用精简搜索入口并正确清理鼠标悬浮状态；更新说明支持安全 Markdown。
- Aurona Manager 增加 Rust target 体积展示、大目录警告和清理二次确认。

## 验证提醒

文件树拖放、编辑器坐标、PTY 初始化、Fliuno 和卡片内标签栏仍应在 Windows Tauri Release 构建中完成最终手工验证。
