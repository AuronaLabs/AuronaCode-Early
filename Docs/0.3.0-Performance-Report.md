# Aurona Code 0.3.0 性能报告

## 方法

- profile：`aurona-desktop-v1`。
- 只比较同一 profile、Release/Debug 模式、OS、架构、CPU 模型和核心信息的数据。
- 完整测试使用 2 次预热和 9 次样本，保存原始样本、均值、截尾均值、P95 和变异系数。
- 综合分以同环境 0.2.10 为 100，使用稳定指标比值的几何平均。CV 高于 15% 的候选项不参与评分。
- v1/v2 历史会迁移保留，但没有 v3 profile 和可比较键的快照不参与排行。
- 原始 JSON 存放在被 Git 忽略的 `.aurona-local/performance/<version>/`。

## 已确认的构建指标

| 资源 | 修改前 | 0.3.0 当前 | 变化 |
| --- | ---: | ---: | ---: |
| Splash CSS | 63.55 kB | 1.36 kB | -97.9% |
| Splash CSS gzip | 10.51 kB | 0.69 kB | -93.4% |
| 主 CSS gzip | 21.13 kB | 22.10 kB | +4.6%，包含 Material 组件与焦点样式扩展 |

Splash 已不再引入 Tailwind 全量扫描样式和品牌 WebFont，使用独立小型 CSS。主 CSS 当前构建为 93.78 kB（gzip 22.10 kB）。上述数字来自本机 `pnpm run build` 产物，不是启动时间基准；不同 Vite 版本和内容哈希不会改变比较口径。
