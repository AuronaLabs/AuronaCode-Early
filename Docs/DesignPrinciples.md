# 设计规范 (Design Specifications)

Aurona Code 遵循极简与功能性优先的 UI 设计原则

## 核心规范

- **功能导向**：UI 元素应直接服务于开发者的代码阅读与操作，避免非必要的装饰性组件
- **视觉层级**：通过内边距、透明度与柔和的色彩对比来划分信息区块，控制界面密度
- **动效克制**：仅在状态切换或反馈时使用基础的 CSS 过渡（如透明度、平移），避免复杂的动画阻碍交互效率

## 技术规范

- **样式框架**：使用 Tailwind CSS 进行组件化样式定义
- **主题化**：所有颜色必须通过 CSS 变量（CSS Variables）引入，禁止在组件层硬编码颜色值
- **图标系统**：统一采用 Tabler Icons
