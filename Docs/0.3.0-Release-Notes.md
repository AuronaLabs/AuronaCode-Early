# Aurona Code 0.3.0 Release Notes

0.3.0 不扩张新功能面，而是对已有编辑、工作台、桌面能力和工程体系进行系统收口。

## 1. 核心架构

- 标签页、活动区、底部面板和文件定位请求统一由 `useWorkbenchStore` 管理。
- `AppServices.start()/dispose()` 统一服务启动、监听器和存储生命周期。

## 2. 编辑器内核

- 增加 revision、client batch ID 和原子 Rope 批量编辑协议。
- 保存校验磁盘指纹，使用临时文件、备份恢复、同步落盘与权限保留。
- 撤销/重做改为有预算的操作历史，并增加未保存文档本地恢复快照。

## 3. 桌面能力与安全

- Tauri API 统一收口到类型化桌面边界，业务模块不再直接导入插件 API。
- 主窗口和 Splash 使用独立 capability，删除不需要的默认权限集。
- Git 阻塞命令进入后台工作线程，搜索和性能任务支持真实取消。

## 4. 命令与工作流

- 菜单、macOS 原生菜单、快捷键和编辑操作共用统一命令 ID。
- `Ctrl/Cmd+Shift+P` 打开紧凑命令面板，只展示真实已注册命令。
- 删除未使用且包含占位 Agent 文案的 TopCommandBar。

## 5. Material UI

- 建立 Canvas、Chrome、Panel、Surface、Overlay、Modal 和 Interactive 语义材质及基础 token。
- 恢复可见键盘焦点，完整遵守 reduced motion。
- 启动页改为独立小型 CSS，不再加载 Tailwind 全量样式与品牌字体。
- 统一玻璃输入框、开关、下拉菜单、列表、搜索与 Git Diff 的材质层级，并为深浅主题独立校准拟物强度。

## 6. 性能与启动

- 取消固定两秒启动等待和强制最大化，主窗口可交互后立即显示。
- 性能记录升级到 schema v3，保存原始样本、截尾均值、P95、CV 和环境可比较键。
- 版本排行使用语义版本顺序，不可比数据不参与评分。

## 7. 稳定性与恢复

- Rust 和前端桌面传输使用可序列化领域错误，transport 不再自动弹出 Toast。
- 高亮器移除 Rust regex 不支持的前瞻表达式，避免特定语言规则初始化 panic。
- 搜索、大文件高亮与同步队列丢弃已知的过期请求。

## 8. 工程质量与跨平台

- Biome 从 153 个错误、31 个警告清理到零。
- 新增 Vitest/React Testing Library、桌面边界检查和 PR CI。
- 版本、锁文件、更新历史、安全文档与架构文档对齐 0.3.0。

## 发布前手工验证

需使用实际 Tauri Release 包验证：IME、快速输入后保存、外部文件冲突、恢复、大文件、Git、PTY、更新器、开发者工具菜单、DPI 与跨平台窗口行为。
